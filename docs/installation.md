# Installation

## Stable release

To install DSBusinessCase, run this command in your terminal:

```sh
uv add CaseStudy_FinanceRocks
```

Or if you prefer to use `pip`:

```sh
pip install CaseStudy_FinanceRocks
```

## From source

The source files for DSBusinessCase can be downloaded from the [Github repo](https://github.com/sharatkatz/CaseStudy_FR).

You can either clone the public repository:

```sh
git clone git://github.com/sharatkatz/CaseStudy_FR
```

Or download the [tarball](https://github.com/sharatkatz/CaseStudy_FR/tarball/master):

```sh
curl -OJL https://github.com/sharatkatz/CaseStudy_FR/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd CaseStudy_FR
uv pip install .
```
